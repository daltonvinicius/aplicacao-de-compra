# Sistema de Controle de Cartão de Crédito

## Descrição
Projeto em Java que simula um **cartão de crédito**, permitindo registrar compras, verificar saldo disponível e gerar um **checklist** detalhado das transações realizadas. O sistema organiza os produtos por valor, mostrando os mais caros primeiro.

## Tecnologias
- Java 11+
- Console para interação com o usuário

## Funcionalidades
- Definir limite do cartão de crédito
- Registrar compras e atualizar saldo automaticamente
- Verificar saldo disponível e aprovar/reprovar compras
- Gerar checklist completo com total gasto e lista de produtos
- Ordenar produtos do maior para o menor preço no checklist


## Observações
- O sistema não permite compras que ultrapassem o limite do cartão.
- Os produtos são listados no checklist em ordem decrescente de preço.
